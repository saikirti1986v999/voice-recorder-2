Frequently Asked Questions

  • How long does my LRS file stay online?

    We keep your LRS file on our server for 24 hours before deleting it. However, if you sign up for an account, we will keep it online for as long as you have an active account or you delete the LRS file yourself.

  • Are there any limitations to sharing?

    You can share just one LRS file at a time using this free tool, and each file must not exceed 100MB in size. Additionally there is a daily limit of 3 LRS file shares. If you wish to share more files or ones larger than 100MB, you can sign up for a free account.

  • Is it safe to share my LRS file online?

    Absolutely! Your LRS file is transferred to our server using an SSL connection, stored using military-grade AES-256 encryption, and shared using a secure shareable link.

  • I have other questions about sharing files online!

    No problem! We have a 24/7 support team to assist you with sharing files online. Just shoot us a message at hello@jumpshare.com and we will get back to you as soon as possible.
